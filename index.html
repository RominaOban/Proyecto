Introducción al HTML

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para crear páginas web. Permite estructurar el contenido mediante etiquetas semánticas, como encabezados, párrafos, listas, imágenes, enlaces y formularios.

A lo largo del tiempo, HTML ha evolucionado desde sus primeras versiones, añadiendo nuevas etiquetas y funcionalidades para mejorar la accesibilidad, la semántica y la interactividad de las páginas web. Actualmente, HTML5 es la versión más moderna y estandarizada, que incluye soporte para video, audio, gráficos y formularios avanzados.
